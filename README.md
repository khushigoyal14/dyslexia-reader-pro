# Dyslexia Assistive Reader

A desktop application built with Python and Tkinter designed to assist individuals with dyslexia by providing customizable typography tools, basic image text extraction, and text-to-speech functionality.

## Current Features
* **Visual Typography Controls:** Real-time modification of text font family styles and font sizes via a sidebar control panel to optimize readability.
* **Document Ingestion (OCR):** Basic integration with the Tesseract OCR engine to extract raw text strings from local image files (.png, .jpg, .jpeg, .bmp).
* **Text-to-Speech Engine:** Synchronous text-to-speech capability using the `pyttsx3` library to read out text content aloud.
